# Transcriptional Disorder

This is the code repository for the paper:

**A Manifold-Based Measure of Transcriptional Entropy for Quantifying Aging in Single Cells**

Preprint: [bioRxiv](https://www.biorxiv.org/content/10.64898/2026.01.24.701460v1)

## Repository Structure

### `allelic-specific-expression/`

Scripts for the two allelic-specific expression (ASE) analyses:

- `ASE_larsson.R` — ASE analysis on the Larsson dataset
- `ASE_ochiai.R` — ASE analysis on the Ochiai dataset

### `Transcriptional_dyscoordination_functions.R`

Core functions for computing transcriptional dyscoordination measures.
